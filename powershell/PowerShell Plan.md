
## 📚 The Book's Progression (so you know what you're walking into)

The book is structured around **"don't memorize — learn to discover."** The core philosophy is `Get-Help` and `Get-Command` — teaching you to fish rather than memorize syntax. Here's the arc:

|Chapters|Theme|Directly Feeds|
|---|---|---|
|1–4|The shell, running commands, discovering commands|Foundation for everything|
|5–8|Pipeline, objects, formatting output|How PowerShell actually thinks|
|9–11|Filtering, sorting, calculated properties|Log analysis — SOC crossover|
|12–14|Remote management, WMI/CIM|Helpdesk video #3, AD work|
|15–17|Scripts, variables, input/output|Your new user automation video|
|18–21|Functions, error handling, debugging|Making scripts production-ready|
|22–25|Modules, profiles, regular expressions|Long game — serious scripting|
|26–28|Final projects and putting it together|Portfolio-ready scripts|
