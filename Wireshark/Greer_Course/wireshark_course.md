## Configuring the Wireshark Interace:

- Enlarge the font (hit +)
- Create a Specific Profile (bottom right corner)
- Create / Modify Columns
    - Example:  Add Delta Time Displayed ( Edit > Preferences)

## ** First Skill **

### Create Custom Filter Rule
ex. TCP SYNS

    - Find Syn Flag in TCP detail info pane below
    - Click on SYN flag > this will point to the syntax of filter (lower left)
    - Copy and paste display filter (tcp.flags.syn) > click + > name and save

### Color Coding

Go to > View > Coloring Rules
*** Important Note ***
They are in order of Priority > Keep bac TCP at the TOP of the list
