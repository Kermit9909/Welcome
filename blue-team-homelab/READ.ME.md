# Blue Team Homelab - Network Security Analysis Practice

## Purpose
This repository documents my hands-on learning journey in network security 
analysis and defensive cybersecurity techniques. All exercises are performed 
on my personal home network with proper authorization.

## Learning Objectives
- Network traffic analysis using Wireshark
- IoT device security assessment
- mDNS reconnaissance and mitigation
- SIEM rule development
- Incident response procedures
- Threat detection methodologies

## Projects

### 1. Home Network IoT Security Assessment
**Objective:** Identify IoT devices on my network using passive monitoring
**Tools:** Wireshark, mDNS analysis
**Skills:** Network reconnaissance, asset inventory, vulnerability identification
**Outcome:** Created network segmentation plan and hardened 5 IoT devices

### 2. DNS Tunneling Detection Lab
**Objective:** Practice identifying DNS-based data exfiltration
**Tools:** Wireshark, custom filters, PCAP analysis
**Skills:** Anomaly detection, protocol analysis, threat hunting
**Outcome:** Developed detection signatures for SIEM implementation

### 3. ICMP Traffic Analysis
**Objective:** Distinguish normal vs malicious ICMP traffic patterns
**Tools:** Wireshark, statistical analysis
**Skills:** Protocol analysis, baseline establishment, IOC identification

## Disclaimer
All activities documented here are performed on networks I own or have 
explicit permission to test. This repository is for educational purposes 
and professional skill development in preparation for a career in 
cybersecurity defense.
